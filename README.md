🌑✨ Opera Home — A Modern Luxury E-Commerce Platform Built with Django
Opera Home is a refined, design-forward e-commerce platform crafted with Django 5, inspired by the aesthetics of quiet luxury.
Engineered for clarity, performance, and elegance, the platform delivers a seamless shopping experience for premium home essentials—wrapped in a bold, black-and-gold visual identity.
🌟 Key Features at a Glance
⚡ Elegant UI with Black & Gold Aesthetic
A fully responsive, minimal, luxury interface tailored for high-end home products.
📦 Complete Product Management (CRUD)
Create, update, delete, and display products with images, categories, and details.
🛒 Persistent Cart System
Each authenticated user automatically receives a personal cart, stored and updated across sessions.
🔐 Django Authentication Integration
Custom User model, clean login page, and secure cart access using LoginRequiredMixin.
📄 Dedicated Static Pages
Styled and responsive About and Contact pages with polished layout and modern copywriting.
🧱 Clean Architecture with CBVs
The project is built entirely using Class-Based Views for scalability and maintainability.
🖼️ Image Upload + Media Handling
Full support for product images with Django’s media pipeline.
🧩 Tech Stack & Tools
Python 3.x
Django 5
SQLite (easily replaceable with PostgreSQL/MySQL)
HTML5 + CSS3
Django Templating Engine
LoginRequiredMixin, CBVs, Namespaced URLs
CustomUser Model (accounts app)
Media + Static File Management
